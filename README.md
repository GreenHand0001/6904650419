# 项目简介

## 核心功能
- 基于YOLOv8的野生动物检测系统，支持9类动物识别：roedeer、wildboar、bird、badger、racoon、Rabenvogel、hare、fox、text
- 支持6800张标注数据集用于训练模型
- 提供完整源码、训练好的模型、数据集及项目运行教程
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持批量检测功能，可查看所有检测结果
- 支持检测结果保存

## 适用场景
- 野生动物监测与研究
- 生态保护与生物多样性调查
- 野生动物行为观察记录
- 野生动物保护区管理
- 野生动物数量统计与分析

## 技术特点
- 采用YOLOv8深度学习框架
- 使用Pycharm/VSCode开发环境
- 基于Python 3.9开发
- 集成OpenCV-Python、PyQt5、Ultralytics、Torch 1.9等技术栈
- 提供图形用户界面，操作简便
